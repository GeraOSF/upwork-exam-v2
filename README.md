# Upwork - Entry Exam

## Setup

Create a virtual environment and activate it:

```bash
python3 -m venv .venv
source .venv/bin/activate or .venv\Scripts\activate
```

## Install

Install the requirements:

```bash
pip install -r requirements.txt
```

## Run

Run the application:

```bash
uvicorn app.main:app --reload
```

## Testing

Run the tests:

```bash
pytest
```
